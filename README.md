# Exercise-03a-Color-and-Particles

Exercise for MSCH-C220, 21 September 2021

The first steps in adding "juicy" features to a simple match-3 game.

## To play

Select and drag tiles using the mouse.


## Implementation

Built using Godot 3.3.3
